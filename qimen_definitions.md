# 奇門遁甲棒球判讀 — 預先註冊定義框架 v1 (2026-06-26)

> 紀律：本檔在「看分析結果之前」鎖定。之後檢定一律 Bonferroni 校正 + 前後半樣本外驗證。
> 誠實前提：把中性的主/客隊對應到用神本身有任意性；本框架=「固定一套標準約定，檢定它能否打敗隨機」，預期多半為 null。

## 1. 起局
- 每場一盤，以**開賽時刻**起時家奇門（已抓 `shiStart`）。
- 首局(NRFI/YRFI)另用**開賽神刻盤**（已抓 `shenkeStart`）。
- 尾局逆轉另參**結束時辰盤**（開賽+160分，已抓 `shiEnd`）。

## 3. 吉凶分級表
- **八門**：吉=開/休/生；平=杜/景；凶=死/驚/傷。（死門=防守→低分傾向）
- **八神**：吉=值符/六合/太陰/九天/九地；凶=白虎/玄武/螣蛇；中=朱雀(口舌)。
- **九星**：吉=天輔/天禽/天心/天任；凶=天蓬/天芮/天柱；中凶=天英。
- **旺衰**：用神五行 vs 月令(季節)→旺/相/休/囚/死。
- **空亡**：用神宮旬空→力量打折。
- **特殊局**：伏吟(主客同宮)=強隊易輸/弱隊翻盤；反吟(主客對沖宮)=場面起伏驚險。

## 4. 各市場判讀規則（operationalized）
**獨贏**：score(宮)=門吉凶+神吉凶+星吉凶+旺衰−空亡；生克方向(主剋客/客剋主/比和)。預測勝方=綜合分高者；平手則看生克。
**讓分(過盤)**：主客力量差距大(強剋+旺衰差大)→熱門大勝(過讓分1.5)；接近→不過盤。
**大小分**：反吟局 / 火重(天英·景門) / 用神逢沖 → 大分；伏吟局 / 死門 / 用神入墓 → 小分。
**YRFI/NRFI(首局)**：開賽神刻盤用神宮門——動門(開/生/景)易得分(YRFI)；靜/凶(休/死)難得分(NRFI)。
**逆轉(尾局)**：伏吟/反吟局，或結束時辰盤用神由旺轉衰/入墓 → 逆轉。

## 5. 每場要算出的特徵（餵檢定）
homeYS=地盤時干宮{門,門吉凶,神,神吉凶,星,星吉凶,旺衰,空亡}；awayYS=天盤時干宮{同}；
shengke(home,away)方向；predWinner(規則)；fuyin/fanyin旗標；值符宮/值使門；
home投手生年干宮門吉凶、away投手生年干宮門吉凶。

## 6. ⚠ 實作前提（須先升級爬蟲＋重抓，約30-60分）
目前 `qimen_data.json` 的 `ganVisible` 把**天盤干/地盤干混在一起**，無法定位「地盤時干宮 vs 天盤時干宮」。需升級 `qimen_scraper.js` 每宮分存：
- **天盤干 / 地盤干 / 暗干**（分開標記）
- **四柱干支**（頁面「干支：年 月 日 時」可抓→得日干/時干）
- **局**（陰陽遁X局，判伏吟/反吟）、**空亡**（Label*_kong）
然後重抓全部 1132 場。重抓後即可計算第5節特徵並跑預先註冊檢定。

## 7. 檢定計畫
獨贏/讓分/大小/YRFI/逆轉 各 1–2 個規則 → Bonferroni(α=0.05/檢定數) + 前半訓練/後半驗證一致才算數。投手生年干組合(user 記得曾做)→ YRFI 列聯表(類別少、先驗低,嚴格校正)。
